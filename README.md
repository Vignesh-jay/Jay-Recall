# Jay Recall

Never Google The Same Thing Twice.

Jay Recall is a personal knowledge management system built for IT professionals, developers, and power users.

## Features

- Knowledge Notes
- Command Vault
- Troubleshooting Library
- Documentation
- Infrastructure Records
- Security Notes
- Checklists

## Search

- Global Instant Search
- Recent Searches
- Favorites

## Data

- Local First Storage
- Import / Export Backup

## PWA

- Installable
- Mobile Optimized
- Offline Ready
